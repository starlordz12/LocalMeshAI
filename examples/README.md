# Examples

Worked example projects for LocalMeshAI. Each example is a saved **operation tree**
(`project.json`) plus any source mesh it references, so you can open it and replay the
edits.

> Examples will be added as the workflows stabilize. To create one: build a project in the
> app, **Save Project**, then copy its `project.json` (and the source mesh) here with a short
> note describing the workflow it demonstrates.

Planned examples:

- `mounting_pad/` — Workflow 1: glue-on mounting pad with screw holes.
- `cooling_vent/` — Workflow 2: annotated rectangular vent cutout.
- `camera_deck/` — Workflow 3: AI-planned aerodynamic camera mounting deck.
